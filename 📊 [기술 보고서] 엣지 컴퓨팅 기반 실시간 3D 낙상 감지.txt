📊 [기술 보고서] 엣지 컴퓨팅 기반 실시간 3D 낙상 감지 파이프라인 구축
1. 개요 (Overview)
본 연구는 고사양 클라우드 서버나 GPU에 의존하지 않고, 초저전력 엣지 디바이스(Coral Dev Board) 단일 기기에서 실시간으로 구동되는 경량화 3D 낙상 감지 시스템을 구축하는 것을 목적으로 한다. 이를 위해 비디오 영상에서 3D 스켈레톤을 추출하여 .npy 형태로 가공하는 전처리 단계부터, 이를 시계열로 학습하는 1D-TCN(Temporal Convolutional Network) 모델 구성, 그리고 최종 Edge TPU 추론을 위한 최적화 파이프라인까지 End-to-End로 설계하였다.

2. 데이터 전처리 및 데이터셋 구축 (Data Preprocessing & Dataset Construction)
NTU RGB+D 데이터셋의 원본 비디오 영상을 시계열 딥러닝 모델이 학습할 수 있는 정형화된 배열(.npy)로 변환하는 과정을 거쳤다.

2.1. 3D 스켈레톤 추출 및 .npy 변환
2D 관절 추출 (YOLOv8-Pose): 영상 내 객체 중 사람(classes=[0])만을 타겟팅하여 17개의 2D 관절 좌표를 추출한 뒤, 하반신 중심점(Pelvis)을 계산하여 동작 인식에 최적화된 13개의 관절 좌표를 선별하였다.

3D 깊이 복원 (Lifting Network): 추출된 2D 좌표를 경량화된 LN(Lifting Network)에 입력하여 Z축(Depth)이 포함된 3D 스켈레톤 데이터로 복원하였다.

데이터 저장: 프레임별 13개의 3D 관절 좌표(X, Y, Z)를 병합하여 파일당 (N, 1, 39) 형태의 차원을 가지는 .skeleton.npy 파일로 저장하였다. (N은 프레임 수)

2.2. 슬라이딩 윈도우 (Sliding Window) 기반 시계열 데이터화
차원 압축: (N, 1, 39) 차원의 데이터를 모델 입력에 맞게 (N, 39)로 압축(Squeeze)하였다.

윈도우 분할: 시계열 동작의 맥락(Context)을 학습하기 위해 Window Size = 60 프레임, Stride = 2를 적용하여 데이터를 슬라이딩 윈도우 방식으로 분할하였다.

제로 패딩 (Zero-Padding): 60프레임 미만의 짧은 영상은 앞부분을 0으로 채워 모델의 입력 규격을 완벽하게 통일하였다.

2.3. Cross-View (CV) 프로토콜 분할
NTU RGB+D의 공식 평가 기준에 따라 카메라 시점(View)을 기준으로 데이터를 분할하였다.

Test Set: 1번 카메라(C001)에서 촬영된 영상

Train Set: 2, 3번 카메라(C002, C003)에서 촬영된 영상
분할된 데이터는 최종적으로 X_cv_train.npy, y_cv_test.npy 등의 텐서 배열로 저장되어 모델 학습의 효율성을 극대화하였다.

3. 시계열 동작 분류 모델 (1D-TCN) 구성
추출된 시계열 3D 스켈레톤 데이터를 분석하여 낙상 여부를 판별하는 핵심 분류기(Classifier)를 설계하였다.

3.1. 1D-TCN 아키텍처 적용
모델 선정: 기존의 2D 이미지 기반 CNN이 아닌, 시간의 흐름에 따른 관절 궤적(Temporal Dynamics) 변화를 추적하는 데 특화된 1D-TCN (1D Convolutional Neural Network 기반) 구조를 채택하였다.

분류 기준: 마지막 출력층에서 60프레임 동안의 동작을 종합하여 '낙상(Fall, A043)'과 '정상(Normal, 그 외 동작)'을 이진 분류(Binary Classification)하도록 구성하였다.

3.2. 공간 정규화 (Spatial Normalization) 전략의 최적화
이중 정규화 문제 해결: 윈도우 텐서 단위로 정규화를 수행할 경우, 낙상 시 사람의 높이(Y축)가 낮아지는 결정적 단서가 훼손되는 현상을 발견하였다.

글로벌 스케일(Global Scale) 적용: 이를 해결하기 위해 영상의 첫 프레임을 기준으로 global_max_dist를 산출하고, 영상 전체에 대해 단일 기준의 스케일 정규화를 수행하여 물리적 높이 변화를 모델이 정확히 인지하도록 개선하였다.

4. 엣지 디바이스 최적화 및 실시간 추론 파이프라인
학습된 모델을 자원이 제한된 Coral Dev Board(Edge TPU)에 탑재하기 위한 최적화 및 추론 로직을 구현하였다.

4.1. INT8 양자화 (Quantization) 및 하드웨어 가속
모델 경량화: FP32 기반의 파라미터를 INT8로 양자화하여 모델 크기를 약 15분의 1 수준인 2.4MB로 극단적으로 압축하였다.

TPU 컴파일: 구글 edgetpu-compiler를 이용해 가중치를 Edge TPU 메모리에 매핑하여 연산 병목을 제거하였다.

4.2. 실시간 추론(Inference) 로직 고도화
실제 엣지 디바이스 구동 시 발생할 수 있는 데이터 누락 및 왜곡을 방어하기 위한 2가지 핵심 로직을 도입하였다.

비율 왜곡 방어 (Letterbox Restoration): 영상을 1:1 정방형으로 강제 리사이즈할 때 발생하는 사람의 신체 비율 찌그러짐을 방지하기 위해 Letterbox 처리를 적용하고, 추출된 관절 좌표를 원본 영상 비율에 맞게 역산하여 복원하는 로직을 추가하였다.

동적 버퍼 유지 (Dynamic Padding): 특정 프레임에서 사람을 일시적으로 놓치더라도(LOST TARGET), 이전 프레임의 뼈대 정보를 버퍼에 복사하여 60프레임의 시계열 흐름이 끊기지 않도록 보장하는 안전장치를 구현하였다.

5. 최종 성능 평가 (Performance Evaluation)
위 파이프라인을 종합하여 Edge TPU 환경에서 최종 테스트를 진행한 결과는 다음과 같다.

정확도 (Accuracy): 98.56% (1717 / 1742)

재현율 (Recall): 82.89% (63 / 76)

정밀도 (Precision): 84.00%

F1-Score: 83.44%

추론 지연(Latency): 프레임당 평균 약 73.5ms (YOLO: ~64ms / LN+1D-TCN: ~9ms)
(Edge TPU 가속을 통해 동작 분류 단계의 연산 시간을 10ms 이하로 단축하여 초당 약 13.6프레임의 실시간 구동 성능 확보)

6. 결론 (Conclusion)
본 연구에서 구축한 파이프라인은 고사양 GPU 환경에서 달성 가능한 98% 이상의 낙상 감지 정확도를 초저전력 엣지 컴퓨팅 환경에서 성공적으로 재현해 냈다. 특히 1D-TCN을 활용한 시계열 3D 관절 분석 파이프라인은 낙상 감지뿐만 아니라 다양한 범용 행동 인식 아키텍처로 확장 가능한 높은 기술적 완성도와 실용성을 증명하였다.
