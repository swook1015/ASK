# 엣지 컴퓨팅 기반의 경량화 3D 낙상 감지 시스템

본 프로젝트는 실시간 낙상 감지 AI 시스템을 위한 통합 파이프라인으로, AI 모델 연구부터 백엔드 서버 연동 및 프론트엔드 시각화까지 전 과정을 포함합니다.

## 📂 브랜치 구조 및 역할 (Branch Structure)

각 브랜치는 개발 단계 및 기능에 따라 다음과 같이 분리되어 관리됩니다.

| 브랜치명 | 역할 및 설명 |
| :--- | :--- |
| **`main`** | 분리 설명을 위한 브렌치 |
| **`ai`** | 낙상 감지 AI 모델 연구 및 테스트 (Phase 1) |
| **`ai2`** | 낙상 감지 AI 모델 개선 및 고도화 (Phase 2) |
| **`fallAI`** | 낙상 감지 AI 모델 핵심 알고리즘 |
| **`backend`** | 서버 시스템 API 및 데이터베이스 연동 |
| **`backend-windows`** | 윈도우 환경 최적화 및 테스트용 백엔드 |
| **`inha`** | 사용자 인터페이스(UI) 및 프론트엔드 웹 시스템 |
